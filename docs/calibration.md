# Calibration Guide

## Why Calibration Matters

This prototype combines direct and proxy measurements.

- DS18B20 temperature sensing is generally reliable for compost temperature
- DHT22 ambient sensing is useful for environmental context
- low-cost moisture sensing should be treated as indicative rather than absolute

## Practical Calibration Notes

1. Verify compost temperature against a trusted thermometer if possible
2. Compare moisture signal values in wetter and drier compost conditions
3. Record baseline readings over several days
4. Look for trends, not just single readings
5. Re-evaluate sensor placement if readings seem inconsistent

## Notes

Compost is a difficult environment for sensors. Moisture, biological activity, and material movement can all affect readings.
